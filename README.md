## Getting Started

1. Install all project dependencies (we use [yarn](https://yarnpkg.com/en/), make sure you have it installed):
    ```
    yarn
    ```
1. Start the development server
    ```
    yarn dev
    ```
1. Visit [http://localhost:3000](http://localhost:3000/)
